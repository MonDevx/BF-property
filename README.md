## Quick start :rocket:

Several quick start options are available:

- Clone the repo: `git clone https://github.com/pimdotzazaza43/BF-property.git`
- Install with yarn install
- Click NPM SCRIPTS dev http://localhost:8070

NPM SCRIPTS

- build webpack
- start webpack
- test
- eject
- deploy
- check
- deploy test
